선택자
- 태그 선택자
 (태그명으로 선택)
 * -> 전체 태그 선택자
 
-클래스 선택자
 .클래스명 (마침표)
 
-아이디
 # 아이디명 (샵)
 
-태그의 style 속성으로 정의하는 방법
 
적용 범위
태그(div) > 클래스 (class) > 아이디 (id) > 태그의 style 속성으로 정의 (html에서 적용)
 
css 적용 우선 순위
(적용 범위가 좁을 수록 우선 순위가 높다)
태그의 속성 style > 아이디 > 클래스 > 태그

우선순위 강제 적용대상
!important -> 우선순위 강제 적용

조상 자손 선택자
왼쪽 - > 오른쪽 
왼쪽으로 갈수록 상위 태그, 오른쪽으로 갈수록 하위 태그)

부모 자식 선택자
바로 위 층위, 바로 아래 층위
부모 > 자식

그룹 선택자
콤마(,)로 구분하여 여러 선택자를 적용

가상 클래스 선택자
	선택자 : 상태값
- 요소의 상태로 선택자
 예1) (체크, 라디오 박스 - > 체크가 되었는지 ?)
  선택자 : checked
  
 예2) 요소에 마우스가 올려 있는 상태인지
  선택자 : hover
 
 예3) 요소가 읽기 전용 상태인가?
 선택자 : read-only
 
 예4) 선택자가 아닌 상태 인가
 선택자1에 선택자2가 포함이 되어 있지 않은 상태인가
 선택자1 : not("선택자2")
 
 예) 첫번째 자식인가
 선택자 : first-child
 
 예) 마지막 자식인가?
 선택자 : last-child
 
 예) 몇번째 자식인가?
 선택자 : nth-child
 nth-child(숫자) -> 번째 요소
 nth-child(수식) -> 수식 계산 후 선택
 2n -> 2, 4, 6, 8, 10... (2의 배수만큼)
 (n은 1부터 시작)
 
 2n+1 ->3, 5, 7, 9, 11 (홀수)
 (n은 0부터 시작)
 
- 가상으로 추가하여 선택하는 선택자
 선택자 : after
  - 선택자의 자손으로 가장 끝에 추가
  
 선택자 : before
  - 선택자의 자손으로 가장 앞에 추가

 형제
 
 인접형제(+)
 선택자1+선택자2 ->선택자1(왼쪽 위치)을 인접 형제로 가진 선택자 2를 선택
 
 일반형제(~)
 선택자1~선택자2 -> 선택자1을 형제(왼쪽 위치)로 가지고 있는 선택자 2를 선택
